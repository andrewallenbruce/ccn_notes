10.1 Hospital Ranges

The last three characters of a Medicaid-Only Hospital CCN represent the facility range. The facility type is encoded in the third character of the last four characters. The table below summarizes the valid facility type codes and their corresponding ranges.

abbr desc range
STC Short-Term Acute Care Hospital 001-099
CH Children's Hospital 100-199
CPH Children's Psychiatric Hospital 200-299
PH Psychiatric Hospital 300-399
IRF Rehabilitation Hospital 400-499
LTCH Long-Term Hospital 500-599
RSVD Reserved for Future Use 600-999

## Hospital Ranges

The last three characters of a Medicaid-Only Hospital CCN represent the facility range. The facility type is encoded in the third character of the last four
characters. The table below summarizes the valid facility type codes and their
corresponding ranges. 

```{r}
#| label: medicaid-hospital-ranges
#| echo: false
#| message: false
ccn::medicaid_ranges |>
  collapse::slt(abbr, desc, range) |>
  collapse::roworder(range) |>
  gt::gt() |>
  gt::tab_options(quarto.disable_processing = TRUE) |>
  gtExtras::gt_theme_nytimes() |>
  gt::tab_style(
    style = gt::cell_text(
      font = gt::google_font(name = "JetBrains Mono"),
      color = "grey60",
      size = "small"
    ),
    locations = gt::cells_body(columns = range)
  ) |>
  gt::tab_style(
    style = gt::cell_text(weight = "bold"),
    locations = gt::cells_body(columns = abbr)
  ) |>
  gt::tab_style(
    style = gt::cell_text(color = "grey50", size = "small"),
    locations = gt::cells_body(columns = desc)
  )
```
