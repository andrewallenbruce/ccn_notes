Medicaid-Only Providers

Published

December 25, 2025

Title XIX-only providers are identified by a 6-position alphanumeric CCN. Positions one and two identify the state in which the provider is located. Position three, an alpha character, identifies the type of facility, by either level or type of care being provided. Positions four, five and six represent a sequence beginning with 001.

Facility Types

The third character of a Medicaid-Only Provider CCN represents the facility type and a unique identifier. The facility type is encoded in the third character of the last four characters. The table below summarizes the valid facility type codes and their corresponding ranges.

abbr desc code
HHA Home Health Agency K
ICF-IID Intermediate Care Facility for Individuals with Intellectual Disabilities G, H
MOH Medicaid-Only Hospital J
NF Nursing Facility A, B, E, F
PRTF Psychiatric Residential Treatment Facility L

Facility Ranges

The last three characters of a Medicaid-Only Provider CCN represent the facility type and a unique identifier. The facility type is encoded in the third character of the last four characters. The table below summarizes the valid facility type codes and their corresponding ranges.

abbr desc range
CH Children's Hospital 100-199
CPH Children's Psychiatric Hospital 200-299
IRF Rehabilitation Hospital 400-499
LTCH Long-Term Hospital 500-599
PH Psychiatric Hospital 300-399
RSVD Reserved for Future Use 600-999
STC Short-Term Acute Care Hospital 001-099
