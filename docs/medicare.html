Medicare

Published

April 10, 2026

Providers and Suppliers paid under Medicare Part A have 6 digit CCNs. The first two characters represent the state in which the provider is located. The last four characters identify the type of facility.

Providers and Suppliers paid under _Medicare Part A_ have __6 digit__ CCNs.
The first two characters represent the state in which the provider is located.
The last four characters identify the type of facility.

## Facility Ranges

```{r}
#| label: medicare-ranges
#| echo: false
#| message: false
ccn::medicare_ranges |>
  collapse::gby(abbr) |>
  collapse::fsummarise(
    desc = collapse::fslice(desc),
    range = toString(range)
  ) |>
  collapse::slt(abbr, range, desc) |> 
  gt::gt() |>
  gt::cols_align("left", columns = "range") |> 
  gt::tab_options(
    table.width = gt::pct(100),
    quarto.disable_processing = TRUE
  ) |>
  gtExtras::gt_theme_nytimes() |>
  gt::tab_style(
    style = gt::cell_text(
      font = gt::google_font(name = "JetBrains Mono"),

      size = "small"

    locations = gt::cells_body(columns = range)
  ) |>
  gt::tab_style(

  ) |>
```
